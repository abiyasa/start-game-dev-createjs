What is CreateJS

http://www.createjs.com/

Multimedia libraries & tools

  • EaselJS: canvas, shape, image, animation
  • SoundJS: pow! kaboom! zing!
  • TweenJS: tweening animations
  • PreloadJS: assets loading
  • Tools for assets
CreateJS is a multimedia library. It's not a full game engine but it has all components to start making (mainly 2D) HTML5 game. Tools for asset work best if you create assets from Adobe Flash.

Why CreateJS?

  • Free & open source
  • Easy to use
  • Mobile/tablet support
  • Good documentation & examples
  • Sponsored by Adobe & Microsoft
  • CDN
Even easier to use if you are familiar with Flash before.
